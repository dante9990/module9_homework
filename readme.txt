Task 3

Данное приложение принимает число от 1 до 10. Если отправлено число вне диапазона, то выводится соответсвующее сообщение.
Если отправленное число соответсвтвует диапазону, то вызывается функция useRequest, которая принимает в себя ссылку с вводим числом, указывающим на количество картинок и коллбэк.
useRequest использует XHR-запрос. Он инициализирует запрос, после происходит проверка на статус сервера.
Если статус 200, то происходит парсинг JSON-строки и JS-объект передается в коллбэк.
displayResult - коллбэк, который при получении API создает в HTML новые элементы с картинками
